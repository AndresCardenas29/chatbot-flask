# Chatbox - V1.0.0

Chatbox desarrollado en Python usando Flask.

### Ejecución del código

> Windows
```properties
python -m virtualenv venv
.\venv\Scripts\activate
python -m pip install -r requirements.txt
``` 

> Linux
```properties
python -m virtualenv venv
./venv/Scripts/activate
python -m pip install -r requirements.txt
```
